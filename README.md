# たびあと

## サービス概要

カップルや友だちとの旅行写真を一緒にアップロードするだけで、自動で「地図へのピン止め」と「時系列のアルバム」を作ってくれるアプリです。
写真に含まれている位置や時間の情報（Exif情報）を活用して、写真の整理だけでなく、「どこに、どれくらい滞在したか」まで可能な限り自動で見えるようにします。


## このサービスへの思い・作りたい理由

私はよく恋人や友だちと旅行に行き、たくさん写真を撮って共有するのですが、写真だけだと「その時どこにいて、どれくらい過ごしたのか」という記憶はどうしても薄れてしまいます。
予定より長居してしまった場所は、普通のアルバムでは見返せない「時間の使い方」や「場所の空気感」まで一緒に振り返ることができる写真には写らない文脈まで楽しめるものがあればより、「あとでログに残るから、ここでも一枚撮っておこう」「移動中の風景も撮っておこう」という風に、このアプリがあることで旅行中にもっと写真を撮りたくなる、そんな体験を作りたいと思ってこのアプリを考えました。

## ユーザー層について

**恋人や友人グループと旅行によく行く人**
それぞれのスマホの中に写真が散らばってしまっていて、それを一つのアルバムにまとめて見返したいと思っている人たち。

**旅行の「行程」や「移動」そのものも楽しみたい人**
有名な観光スポットでの記念撮影だけでなく、「どういうルートで回ったか」「どのくらい歩いたか」という旅のログ（記録）を残すことに面白さを感じる人たち。

## サービスの利用イメージ

* 帰りの電車の中や帰宅後に、参加したメンバーがそれぞれのスマホで撮った写真をアプリにアップロードします。
* 「この時こんなの撮ってたんだ！」という発見があり、同じ時間を過ごしていても見ている景色が違っていたことに気づき、相手の目線を通してもう一度旅を楽しむことができます。

**写真から自動計算される「滞在時間」で、旅の記憶が鮮明によみがえる**
* アップロードされた写真の位置情報をもとに、撮影場所が近い写真を自動でグループ化（ひとまとまりに）します。そのグループごとに、最初と最後に撮った写真の撮影時刻から、おおよその滞在時間を計算します。
* 地図上の写真と一緒に「滞在時間：約2時間 (13:00頃〜15:00頃)」のように表示され、当時の空気感や過ごした時間の流れを直感的に振り返ることができます。
* 「写真を撮る」という行為がそのまま旅の記録になるため、旅先での能動的なアクションを促すきっかけにもなります。

**他の旅行者の参考になる**
* 同じ旅行先を周ろうと考えてる人に対して観光場所の滞在時間がわかり、旅行プランの手助けとなる。

## ユーザーの獲得について

**ログインなしで「人の旅」を見れるように**
いきなり登録させるのではなく、まずはログインなし（ゲスト状態）でも、公開されている他の人の旅行記やアルバムを見れるようにして、「自分もこんな風に残したい」と思ってもらえるようにします。

**「共同編集」で自然に仲間を招待**
アルバムを作る時に「一緒に写真を入れる」ために、パートナーや友だちをLINEなどで招待する流れを自然に作ります。招待された側もそのまますぐに参加できるようにし、ユーザーの輪を広げます。

## サービスの差別化ポイント・推しポイント

### 競合サービスとの差別化

**LINEのアルバム機能との違い:**
LINEでのアルバム作成は共有という点で最高ですが、「たびあと」は「旅のストーリーを再現すること」に特化しています。ただ写真を並べるだけでなく、地図や滞在時間を組み合わせることで、過去の旅行を追体験できるものにします。

**notabiなどの旅行記録アプリとの違い:**
notabiなども旅行後の記録を残せますが、たびあとは「Exif情報による自動解析（滞在時間の可視化）」と「グループでの共同作成」という点が違います。
「写真を放り込むだけで、自分たちがどれくらいそこにいたのかという事実（ログ）が勝手に出来上がる」という手軽さを重視しています。

### 本サービスの推しポイント

**写真データの「副産物」を活かす:**
ユーザーは何も意識せず、ただ写真をアップロードするだけでOKです。写真に元々ついているExif情報から「滞在時間」や「移動ルート」を表示します。面倒な入力作業は一切なしで、価値ある記録が残せる点が一番の強みです。また、その滞在時間データは、次に同じ場所へ行く他の人にとっても参考情報になります。

## 機能候補

### MVP

**1. ユーザー認証・アカウント機能**
* **会員登録・ログイン機能**: データ保存・復元用
* **ゲスト利用機能**: 登録しなくても、他の人の公開アルバムは見れるようにします。

**2. アップロード機能**
* **写真アップロード機能**: 複数枚を一気にアップロードできるようにします。
*  ・データ公開設定: 保存されたアルバムはデフォルトでは非公開。ユーザーが意図して操作しない限り公開されない。
* **自動解析・グループ化機能**: アップされた写真の日付や位置情報（Exif）を自動で読み取ります。さらに、撮影場所や時間が近い写真を自動的に「1つのスポット」としてグループ分けします。

**3. アルバムを見る機能**
* **タイムライン表示**: 写真を時系列順に並べて表示します。
* **マップ表示**: 地図上の撮影した場所に写真をピン留めして表示します。

### 本リリース

* **滞在時間表示**: 同じ場所で撮った最初と最後の写真の時間差から、簡易的な滞在時間を計算して表示します。
* **グループの手動編集機能**: 自動でグループ分けされたスポットを、手動で結合したり分割したりできる機能を追加します。これにより、機械的な判定が実際の感覚とズレていた場合でも修正が可能になります。
* **グループ招待機能**: 作ったアルバムに友だちを招待して、共同編集メンバーに追加します。
* **移動ルートの線画**: 点（スポット）だけでなく、移動した軌跡を線で結んで、どう移動したかわかるようにします。
**ソーシャル・共有機能**
* **公開設定**: プライベートな思い出として楽しむだけでなく、「おすすめの観光ルート」として全体に公開できるようにします。
* **リアクション機能**: アルバム内で「いいね」やコメントができるようにして、アプリ内での会話を楽しめるようにします。
